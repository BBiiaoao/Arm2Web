# Arm2Web
这是一个监听充电桩电流电压的前端动态数据展示页面,前端每隔一定时间展示从后端的实时数据，并将这个数据以图表的简单可视化方式展示给用户看(主要是给让我帮忙写这个页面的同学的项目答辩老师看)
## 前端
>
基于vue-cli3搭建的vue项目，数据展示使用的是echarts组件，数据请求采用axios
>
